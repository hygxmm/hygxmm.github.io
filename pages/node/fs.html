# Node.js 文件系统

Node.js提供一组类似UNIX(POSIX)标准的文件操作API.Node导入文件系统模块的语法如下:

const fs = require('fs');
1

# 同步和异步

Node.js文件系统模块中的方法均有异步和同步版本.异步的方法函数最后一个参数为回调函数,回调函数的第一个参数包含了错误信息.建议使用异步方法,比起同步,异步方法性能更高,速度更快,而且没有阻塞.
