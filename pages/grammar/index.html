# Markdown 语法

# 标题

想要设置为标题的文字前加#来表示

一个#是一级标题,二个#是二级标题,以此类推.支持六级标题,分别对应h1~h6标签

示例:

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

# 字体

  • 加粗

要加粗文字左右两边分别用两个**号包起来

示例:

**这是加粗的文字**
1

效果:

这是加粗的文字

  • 倾斜

要倾斜的文字左右分别用一个*号包起来

示例:

*这是倾斜的文字*
1

效果:

这是倾斜的文字

  • 斜体加粗

要倾斜和加粗的文字左右分别用三个*号包起来

示例:

***这是斜体加粗的文字***
1

效果:

这是斜体加粗的文字

  • 删除线

要加删除线的文字左右分别用两个~~号包起来

示例:

~~这是加删除线的文字~~
1

效果:

这是加删除线的文字

  • 底纹

示例:

`带底纹的文字`
1

效果:

带底纹的文字

# 引用

在引用的文字前加>即可。引用也可以嵌套，如加两个>>三个>>>

示例:

>这是引用的内容
>>这是引用的内容
>>>这是引用的内容
