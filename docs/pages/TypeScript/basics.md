## 1. 如何快速搭建 TypeScript 学习开发环境？

1. 安装 TypeScript

```sh
npm i -g typescript
```

2. 如果想在 node 中直接运行 ts 代码,需要安装 ts-node

```sh
npm i -g ts-node
```
